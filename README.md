# Final_LeguajesFormalesAutomatas
Chomsky Classifier AI es una aplicación interactiva desarrollada con Streamlit que permite analizar, clasificar y visualizar:
- Gramáticas (Tipos 0–3)
- Autómatas (DFA, NFA, PDA, TM)
- Árboles de derivación
- Conversión de modelos (Regex → DFA → Gramática, GLC → PDA)
- Comparación heurística entre gramáticas
- Tutor interactivo con ejercicios
- Generación de reportes PDF automáticos

Requisitos del sistema
Software necesario:
- Python 3.8+
- Navegador web (Chrome, Edge, Firefox)

Dependencias:
- streamlit
- automata-lib
- graphviz
- reportlab
- pandas

Instalación:
pip install streamlit automata-lib graphviz reportlab pandas

Cómo iniciar la aplicación:
- Abrir terminal o CMD
- Posicionarse en la carpeta del proyecto

Ejecutar:
streamlit run app.py

La app se abre automáticamente en:
http://localhost:8501
